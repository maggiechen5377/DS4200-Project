// Data Introduction

The Dataset

// Source

For this analysis, we are using the Video Game Sales dataset from Kaggle, originally compiled by Gregory Smith (GregorUT). The dataset is scraped from VGChartz.com using Python's BeautifulSoup library and includes sales information for video games that sold more than 100,000 copies. The original dataset contained 16,598 observations across 11 variables, and we manually reduced it to approximately 8,300 records by editing the CSV file directly to create a more manageable dataset for analysis.

// Attributes

The dataset includes the following 11 variables across all records:

Rank Overall sales ranking
Name Game title
Platform Console (PS4, Wii, Xbox, etc.)
Year Release year (1980 to 2016)
Genre Action, Sports, Shooter, RPG, etc.
Publisher Publishing company
NA_Sales North America (millions)
EU_Sales Europe (millions)
JP_Sales Japan (millions)
Other_Sales Rest of world (millions)
Global_Sales Worldwide total (millions)

// Preprocessing

We applied the following cleaning steps before analysis: rows with missing Year values were dropped, missing Publisher values were filled with "Unknown", and Year was cast from float to integer. We also derived a Top_Region column that identifies the region with the highest sales for each game. Platforms with abbreviated codes were renamed for readability, for example "2600" became "Atari 2600", "GEN" became "SEGA Genesis", and "XOne" became "Xbox One".

// Why This Dataset

This dataset is well suited for gaming market analysis because it provides both descriptive attributes (genre, platform, publisher) and quantitative sales measures across four distinct geographic regions. The combination of categorical and continuous variables supports a wide range of visualization types, from distribution comparisons to flow diagrams.

One limitation is that it only covers physical and select digital sales tracked by VGChartz, so it likely underrepresents digital-only titles, especially those released after 2012 when digital distribution became widespread. Additionally, the coverage thins out after 2015, which is reflected in the sharp drop in 2016 totals.
